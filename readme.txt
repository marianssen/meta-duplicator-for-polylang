== Description ==

**Easily copy post meta and ACF fields across Polylang translations.**
This lightweight plugin adds a meta box to your post edit screen. With a single click, copy all custom fields (including ACF content) from a translated post into the current one.

= Features =
* Easy to use, no setup needed
* One-click sync between translations
* Works with any public CPT and all custom ACF and meta fields
